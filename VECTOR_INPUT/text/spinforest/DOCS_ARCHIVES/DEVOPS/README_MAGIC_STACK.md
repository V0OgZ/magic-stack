## API Endpoints

- **GET /api** - Documentation API
- **POST /api/interstice/upload** - Upload d'entité
- **POST /api/interstice/download** - Download d'entité
- **POST /api/magic/cast** - Exécution de formule
- **GET /api/magic/health** - État du service
