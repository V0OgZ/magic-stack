- **Géométrie adaptative** = Euclide pour débutants, Non-Euclide pour experts

```
🌀 GÉOMÉTRIES POSSIBLES :
├── 📐 EUCLIDIENNE (Débutants)
│   └── Hexagones plats traditionnels
```

**GRUT définit la hiérarchie :**

```

```
