## 🌟 CONCEPT GÉNÉRAL

**AVALON BOARD GAME** est un jeu de stratégie coopératif où les joueurs incarnent des **Architectes Magiques** explorant les 9 districts d'Avalon pour éveiller les essences et sauver le royaume des corruptions temporelles.

### 🎯 **OBJECTIF**
Collaborer pour éveiller **6 Essences Légendaires** et activer le **Portail Bootstrap** avant que les **Corruptions Temporelles** ne détruisent Avalon !

### 👥 **JOUEURS**
- **2-6 joueurs** (idéal : 4 joueurs)
- **Âge** : 12+ 
- **Durée** : 90-120 minutes
- **Type** : Coopératif avec éléments compétitifs

## 🎮 MÉCANIQUES DE JEU

### **🎯 TOUR DE JEU**

#### **PHASE 1 : ACTIVATION**
1. **Mouvement** : Se déplacer de 1-2 districts
2. **Action Principale** : Choisir parmi :
   - 🔍 **Explorer** un district
   - 🎴 **Jouer** une carte Grammaire Temporelle  
   - ⚔️ **Combattre** une corruption
   - 🤝 **Coopérer** avec un autre héros

#### **PHASE 2 : ÉVÉNEMENT**  
- Piocher une carte Événement du district actuel
- Résoudre l'effet (positif ou négatif)

#### **PHASE 3 : CORRUPTION**
- La corruption progresse sur la piste
- À certains seuils : effets négatifs globaux

### **🔮 SYSTÈME DE GRAMMAIRE TEMPORELLE**

#### **FORMULES MAGIQUES**
Les joueurs combinent des symboles pour créer des sorts :

```
⊙ (Sol) + ψ (Intention) + ⟶ (Projection) = SORT BASIQUE
⊙ + † (Puissance) + ψ + Δt (Temps) + ⟶ = SORT AVANCÉ
```

#### **EXEMPLES DE SORTS**
- **⊙ + ψ_MOV ⟶** : Téléportation entre districts adjacents
- **† + ψ_HEAL + ⟶ Δt+1** : Soigner tous les héros (effet retardé)  
- **⊙ + ψ_REVEAL + Π + ⟶** : Révéler toutes les cartes Événement d'un district

### **💎 SYSTEM D'ESSENCES**

#### **6 ESSENCES À ÉVEILLER**
1. **🔷 Cristal Hexagone** - Améliore les mouvements
2. **📜 Grimoire Vivant** - Bonus cartes Grammaire
3. **🌀 Portail Bootstrap** - Victoire si toutes les autres éveillées
4. **🎤 Essence Nelson** - Commentaires bonus (relances)
5. **⚔️ Arsenal Légendaire** - Armes puissantes
6. **🧙‍♂️ Essence Merlin** - Magie avancée

#### **CONDITIONS D'ÉVEIL**
- Rassembler 3 jetons spécifiques dans le bon district
- Réussir un test de Grammaire Temporelle
- Coopération de 2+ héros minimum

### **⚠️ SYSTÈME DE CORRUPTION**

#### **SOURCES DE CORRUPTION**
- Échecs critiques sur les dés
- Cartes Événement négatives
- Trop de temps dans la Crypte-AR
- Progression automatique chaque tour

#### **EFFETS DE CORRUPTION**
- **Niveau 1-3** : Malus mineurs aux dés
- **Niveau 4-6** : Districts bloqués
- **Niveau 7-9** : Héros corrompus (joueur éliminé)
- **Niveau 10** : **DÉFAITE TOTALE**

## 🏆 CONDITIONS DE VICTOIRE & DÉFAITE

### **🌟 VICTOIRE COOPÉRATIVE**
- Éveiller les **6 Essences** ET activer le **Portail Bootstrap**
- Tous les héros survivent à la purification finale
- **Bonus** : Victoire en moins de 20 tours = "Maîtres d'Avalon"

### **💀 CONDITIONS DE DÉFAITE**
- Corruption atteint le niveau 10
- Plus de 3 districts totalement corrompus  
- Tous les héros éliminés

## 🎲 VARIANTES & EXTENSIONS

### **📈 MODES DE DIFFICULTÉ**

#### **🌱 NOVICE** (Découverte)
- Corruption progresse tous les 2 tours
- +1 carte Grammaire par héros au début
- Pas de cartes Corruption dans les 5 premiers tours

#### **⚔️ HÉROÏQUE** (Standard)  
- Règles normales
- Équilibre défi/plaisir optimal

#### **🔥 LÉGENDAIRE** (Expert)
- Corruption progresse chaque tour
- -1 carte Grammaire par héros
- Événements plus difficiles

#### **💀 CAUCHEMAR SID** (Masochiste)
- Corruption double vitesse
- Pas de relances autorisées
- 5 essences seulement pour gagner (mais plus difficiles)

### **🎪 VARIANTES AMUSANTES**

#### **🎤 MODE NELSON COMMENTATEUR**
- Un joueur joue Nelson et commente toutes les actions
- Bonus/malus selon la qualité des commentaires
- *"OH LA LA ! What a SPECTACULAR échec critique !"*

#### **🎯 MODE SID ARCHITECTE**  
- Le joueur Sid peut réorganiser le plateau une fois par partie
- Mais doit justifier en termes de "design de jeu"
- *"Ce district serait mieux ici pour l'équilibrage !"*

#### **🌀 MODE COOPÉRATIF COMPLET**
- Pas de figurines individuelles
- L'équipe contrôle tous les héros ensemble
- Décisions par consensus obligatoire
