Par MEMENTO

LES 6 DIMENSIONS EXPLIQUEES SIMPLEMENT:

1. X,Y,Z = Position normale (comme tous les jeux)
   - X = gauche/droite
   - Y = haut/bas  
   - Z = altitude

2. T = Temps
   - Present = maintenant
   - Passe = ce qui s'est deja passe
   - Futur = ce qui va arriver

3. C = Causalite (cause et effet)
   - Si je lance une boule de feu, elle fait des degats
   - Les degats peuvent declencher d'autres effets
   - Comme des dominos qui tombent

4. Q = Quantique (plusieurs possibilites)
   - Comme le chat de Schrodinger
   - Plusieurs etats possibles en meme temps
   - L'observation fixe le resultat

EXEMPLES CONCRETS:

Hero Teleport:
- Change X,Y (position)
- Peut changer T (voyage dans le temps)
- Cree une bifurcation C (nouvelle timeline)

Carte "Echo Temporel":
- Rejoue une action du passe (dimension T)
- Dans le present (dimension T actuel)
- Peut creer des paradoxes (dimension C)

Mini-Map 6D:
- Montre position normale (X,Y,Z)
- Slider pour naviguer dans le temps (T)
- Indicateurs de bifurcations (C)
- Probabilites quantiques (Q)
